# 🚀 Step-by-Step Setup

## Step 1: Install Python

Download Python:

https://www.python.org/downloads/

During installation:

✅ Select "Add Python to PATH"

Verify Installation:

```bash
python --version
```

---

## Step 2: Install VS Code

Download:

https://code.visualstudio.com/

Install:

- Python Extension

---

## Step 3: Create Project Folder

Create folder:

```text
education-agent
```

Inside folder create:

```text
app.py
README.md
requirements.txt
```

---

## Step 4: Install Streamlit

Open terminal inside VS Code.

Run:

```bash
pip install streamlit
```

If pip does not work:

```bash
python -m pip install streamlit
```

---

## Step 5: Verify Streamlit Installation

Run:

```bash
python -m streamlit --version
```

---

## Step 6: Add Source Code

Copy the Streamlit source code into:

```text
app.py
```

---

# ▶️ Run the Application

Open terminal.

Run:

```bash
python -m streamlit run app.py
```

---

# 🌐 Application URL

After running, browser opens automatically:

```text
http://localhost:8501
```
